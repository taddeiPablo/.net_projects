# .NET Projects

Colección de proyectos desarrollados con el framework .NET, incluyendo aplicaciones de gestión empresarial y ejercicios de patrones de diseño en C#.

## Proyectos

### 1. Estacionamiento

**Descripción:** Aplicación de escritorio desarrollada en C# con Windows Forms para la administración integral de estacionamientos. Permite gestionar espacios disponibles, registrar entrada/salida de vehículos y generar reportes de ocupación.

**Tecnologías:**
- C#
- Windows Forms (.NET Framework)

**Funcionalidades principales:**
- Registro de vehículos
- Control de espacios disponibles
- Cálculo automático de tarifas
- Historial de movimientos

---

### 2. Practica_Patrones_en_CSharp

**Descripción:** Colección de ejercicios prácticos que demuestran la implementación de patrones de diseño en C#. Este proyecto sirve como referencia educativa para entender conceptos fundamentales de arquitectura de software.

**Patrones implementados:**
- Creacionales: Factory, Abstract Factory, Builder, Prototype, Singleton
- Estructurales: Adapter, Bridge, Composite, Decorator, Facade, Proxy
- De comportamiento: Observer, Strategy, Command, Memento, DAO

**Tecnologías:**
- C#
- .NET Framework / .NET Core

**Nota:** Continuación disponible en el proyecto "Practicas"

---

### 3. QuarkCotizador

**Descripción:** Sistema de gestión empresarial desarrollado para administrar inventario y ventas en tiendas de ropa mayorista. Realiza cálculos automáticos de precios según tipo de cliente, volumen de compra y categoría de prenda.

**Tecnologías:**
- C#
- Windows Forms / ASP.NET

**Funcionalidades principales:**
- Gestión de catálogo de prendas
- Control de inventario y stock
- Cálculo dinámico de precios por cliente
- Generación de cotizaciones y órdenes de compra
- Reportes de ventas y análisis de rotación

---

## Requisitos
